# 全体設計書の宿題一覧

ADR 0001〜0006 の本文とペルソナレビュー（PR #1・#3・#5・#7・#9・#11 のコメント記録）で全体設計書・後続文書に委任された事項の集約。**全体設計書の各文書を起筆する際は、まずこの一覧から該当項目を引き取り、解決した項目には解決先（文書・節）を記入する**。全項目が解決先を持ったらこの文書の役目は終わる。

凡例: ⬜ 未解決 / 出典は ADR 番号（本文）または PR 番号（レビュー申し送り）

## 保守ルール

- 新しい委任が ADR・レビューで発生したら本一覧に追記する（発生 PR で）
- 項目を解決した PR は該当行を「✅ 解決 → <文書#節>」に書き換える（消さない——どこで解決したかが追跡情報）
- **実測待ちの数値・手段は各設計文書の実測待ち一覧（例: architecture.md §9）が管轄する**。機構が設計文書で確定し実測待ち一覧に載った時点で本一覧では解決扱いとし、二重追跡はしない
