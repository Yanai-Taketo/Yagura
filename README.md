# CLA signatures (do not merge into main)

This orphan branch stores contributor CLA signatures for **CLA Assistant Lite**
(see `.github/workflows/cla.yml` on `main`). The action records signatures in
`signatures/version1/cla.json` on this branch. This branch is intentionally
separate from `main` so signature commits do not touch `main`
(conventions.md: no direct pushes to `main`).
